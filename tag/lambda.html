Introduction to AWS Lambda

Published: Tue 23 July 2019
Updated: Tue 23 July 2019
By Robert Koch

In Upcoming talks.

tags: lambda aws compute cloud

Wednesday July 23rd, 2019 - 5:30pm EST

Zoom meeting details will be here 30 minutes before talk

Presented by Robert Koch

Lambda is one of the core services from AWS (Amazon Web Services). Robert will introduce you to the workflow you can use to develop compute functions that usually ties components together whether they're on AWS or not. AWS competitors, Azure Functions and Google Cloud Functions, both have similar development workflow and cost as Lambdas. Robert will be using Python to demonstrate. No prior knowledge of AWS or Lambdas needed.

Presented in American Sign Langauge (ASL). We will have a American Sign Language interpreter also available to voice and take in questions from our non-signers audience.
